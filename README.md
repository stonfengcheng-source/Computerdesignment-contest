# 🌐 深蓝卫士——基于多模态语义感知与图谱风险溯源的游戏净化平台

> 🚀 面向网络环境治理的智能分析平台
> 🎯 聚焦“污染检测 + 溯源分析 + 证据留存 + 决策辅助”全流程闭环

---

## 👋 项目简介

大家好，欢迎来到本项目！

我们是一支参加计算机设计大赛的团队，

在当前网络环境中，**阴阳怪气、地域攻击、隐性黑话等文本污染形式愈发隐蔽**，传统关键词匹配方法已难以应对。本项目通过**深度学习 + OCR + 图算法 + 数据分析**，实现从**检测 → 理解 → 溯源 → 处置建议**的一体化解决方案。

---

## ✨ 核心亮点（Highlight Contributions）

🔍 **1. 语义级文本污染识别（突破关键词局限）**

* 基于 **RoBERTa** 语义模型
* 可识别：

  * 阴阳怪气表达
  * 地域攻击
  * 网络黑话
  * 隐性攻击语义
* 👉 实现从“字面检测”到“语义理解”的跃迁

---

🎥 **2. 视频文本自动提取（跨模态分析）**

* 集成 **RapidOCR**
* 支持：

  * 视频帧文本提取
  * 弹幕/字幕分析

---

🧠 **3. 污染传播溯源图算法**

* 构建文本传播关系图

* 支持识别：

  * 🧩 单局污染源头（Source）
  * 🔥 核心污染节点（Core Spreaders）
  * 🌊 次级传播节点（Secondary Spreaders）

* 👉 从“检测污染”升级为“分析污染传播机制”

---

📊 **4. 可审计分析报告**

* 自动生成完整分析报告
* 包含：

  * 原始文本证据
  * 模型分析结果
  * 污染传播路径
* 提供：

  * 📄 多方共享能力
  * ⚖️ 惩罚建议辅助决策

---

🗂️ **5. 自建高质量数据集**

* 构建专属文本污染数据集
* 覆盖多种复杂语义场景
* 👉 支持模型持续优化与研究拓展

📩 *数据集获取方式：可私信联系团队*

---

## 🏗️ 系统架构（System Overview）

```
数据输入（文本 / 视频）
        ↓
OCR提取（RapidOCR）
        ↓
语义分析（RoBERTa）
        ↓
污染识别结果
        ↓
图建模（传播关系）
        ↓
污染源溯源分析
        ↓
报告生成 & 决策建议
```
